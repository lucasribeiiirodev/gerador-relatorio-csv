# 📊 Gerador de Relatório CSV

Aplicação desenvolvida em **Python** com **Tkinter** para gerar relatórios a partir de arquivos CSV de vendas.

O sistema permite que o usuário selecione um arquivo CSV e visualize informações como:

- 📦 Total de vendas
- 🏆 Produto mais vendido
- 📈 Média de vendas

## 🚀 Funcionalidades

- Selecionar um arquivo CSV
- Ler e processar os dados
- Calcular o total de vendas
- Identificar o produto mais vendido
- Calcular a média de vendas
- Exibir os resultados em uma interface gráfica

## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/lucasribeiiirodev/gerador-relatorio-csv.git
```

2. Acesse a pasta do projeto:

```bash
cd gerador-relatorio-csv
```

3. Execute:

```bash
python main.py
```

4. Selecione um arquivo CSV para gerar o relatório.
